# EC2 Monitoring Assignment

## Objective
Learn cloud setup, shell scripting, automation and monitoring of EC2 instances.

## Files
- monitor.sh
- monitor_logs.txt

## Features
- Shows running services
- Shows CPU usage
- Shows memory usage
- Shows disk usage
- Saves output to log file
- Runs automatically using cron

## Commands Used
./monitor.sh
./monitor.sh >> monitor_logs.txt
crontab -l
git push
